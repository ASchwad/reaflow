# Refs
You can get a reference to the `Canvas` instance to perform various
functions externally.

```jsx
export const MyCanvas: FC = () => {
  const ref = useRef<CanvasRef | null>(null);

  useEffect(() => {
    // Refs give you ability to do things like:
    // ref.current?.centerCanvas()
  }, [ref]);

  return <Canvas ref={ref} />;
};
```

The `ref` will expose extra functions:

- `centerCanvas` - Recenter the canvas programmatically
- `zoomIn` - Zoom in on the canvas
- `zoomOut` - Zoom out on the canvas
- `fitCanvas` - Fit and center the canvas to the view port
- `setZoom` - Manually set a zoom threshold
